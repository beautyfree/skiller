# Skiller Icon Brief

Use this prompt to generate a clean standalone icon source for `Skiller`.

## Goal

Create a production-ready app icon that keeps the current rabbit identity but looks sharper and more distinctive for desktop app stores and taskbars.

## Prompt

```text
Design a clean standalone app icon for a desktop developer tool named “Skiller”.

Visual direction:
- Minimal rabbit symbol as the core mark (sleek, geometric, confident)
- Dark premium background (near-black / deep graphite)
- Subtle indigo-blue accent lighting
- High contrast white rabbit silhouette with soft depth, not flat clipart
- Modern, technical, elegant, calm
- Fits productivity/developer tooling, not gaming or cartoon branding

Composition:
- Single centered icon in a rounded square container
- No text, no wordmark, no UI screenshot, no extra objects
- Keep shape recognizable at small sizes (16x16, 32x32)
- Clear silhouette and readable ears/head profile

Style constraints:
- Avoid cute/cartoon style
- Avoid heavy gradients and visual noise
- Avoid neon cyberpunk overload
- Avoid hard shadows that blur at small sizes

Output requirements:
- Main output: 1024x1024 PNG with transparent outer area if possible
- Also provide vector SVG version of the rabbit mark (separate from container)
- Keep edges crisp and anti-aliased
```

## Acceptance Checklist

- Rabbit silhouette is immediately recognizable at 32x32.
- Icon remains clean on dark and light desktop backgrounds.
- Symbol is unique and does not resemble competitor logos.
- SVG source exists for future scaling and animation.
