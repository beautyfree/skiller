Product tour

Eight slices of the app: visibility, skills, marketplace, settings, sources, sync, search, and paths.

Dashboard

System-wide view of detected agents, how many skills each one uses, search and filters — no more guessing what lives where.

Skills Manager

Browse every install, see which agents consume a skill, edit SKILL.md inline with markdown preview, sync or remove in one click.

Marketplace

Search skills.sh and ClawHub in-app, preview targets and repo metadata, install without touching the filesystem by hand.

Settings

Theme, accent, window blur (macOS), language, close behavior, cache — and App Updates with restart-and-install when a new build is ready.

Skill sources

Import from a Git URL or a local folder to register extra repos, browse their skills, and keep them in sync alongside marketplace installs.

Multi-agent sync

See which agents consume each skill, then push or remove updates across those targets in one flow — no copy-paste between install trees.

Global search

One field to filter skills, agents, and sources — jump to an install or open the editor without digging through folders.

Paths at a glance

Inspect where each skill lands on disk for every detected agent before you sync — fewer surprises when paths differ by tool.

We love clarity.
